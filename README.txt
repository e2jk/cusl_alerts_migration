Le script nécessite python version 3.4+ et la bilbiothèque "pandas".

Le fichier des alertes doit être déposé dans le folder "Data".
--> Il ne doit y avoir que ce fichier dans le folder.
--> Ce fichier doit être un fichier excel (".xlsx").

Le résultat du script va s'enregistrer dans le folder "Script\output\".
--> S'il n'existe pas, il se créera automatiquement.

Le script peut prendre plusieurs minutes à s'executer.
